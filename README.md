# Run PyOPIA on EDITO
Analyse ocean particle images with [PyOPIA](https://pyopia.readthedocs.io). 
A small complete dataset and example of running PyOPIA on the [EDITO Datalab](https://datalab.dive.edito.eu/) platform.


## Run on EDITO
Either find the tutorial in the list on EDITO, or start a "jupyter-tensorflow" service with Python version 3.12.

## Run locally
To run locally (outside EDITO), install (uv)[https://github.com/astral-sh/uv], and then start Jupyter lab with

`uvx --python 3.12 jupyter lab`

Then open and run the notebook.
